# 🚀 Formulário Interativo de Login e Cadastro

Este é um projeto de interface web focado em uma experiência de usuário (UX) fluida e moderna, apresentando um formulário de Login e Cadastro com transições suaves e design responsivo.

## 💡 Sobre o Projeto

O objetivo deste projeto foi criar uma interface de autenticação atraente e funcional. Através de manipulação de estados no CSS e interações simples via JavaScript, o usuário pode alternar entre as telas de "Fazer Login" e "Cadastrar" de maneira elegante, sem necessidade de recarregar a página.

### 🌟 Principais Funcionalidades
* **Transição Fluida:** Animações CSS que proporcionam uma transição suave entre os dois estados do formulário.
* **Design Responsivo:** Layout adaptável para diferentes tamanhos de tela.
* **Interface Limpa:** Uso de hierarquia visual clara e paleta de cores moderna.
* **Simplicidade:** Código limpo e de fácil manutenção.

---

## 🛠 Tecnologias Utilizadas

Este projeto foi construído utilizando tecnologias base da web:

* **HTML5:** Estruturação semântica do conteúdo.
* **CSS3:** Estilização, animações e responsividade (Flexbox/Grid).
* **JavaScript:** Lógica de alternância entre estados do formulário.
